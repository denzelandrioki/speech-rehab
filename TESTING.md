# Speech rehab — ручное тестирование (MVP)

Тестовая сборка для обратной связи, не из Google Play. Установка только из APK.

## Как установить APK

1. На телефоне: **Настройки → Безопасность** (или «Приложения») — разрешите установку из неизвестных источников для браузера / файлового менеджера, с которого открываете APK.
2. Скопируйте APK на устройство и откройте файл.
3. Если уже стояла **другая** подписанная версия — сначала удалите старое приложение (или установите поверх только при том же ключе подписи).

## Где лежит APK после сборки

Стандартный вывод Gradle для модуля `app` (папка **`app/build/`**, не корень репозитория):

- debug: `app/build/outputs/apk/debug/app-debug.apk`
- release: `app/build/outputs/apk/release/app-release.apk`

Сборка **release** подписана **debug-keystore** (удобно для рассылки друзьям; не для Play Store).

Если у вас в папке оказался только `app-release-unsigned.apk`, в `app/build.gradle.kts` для `release` должно быть `signingConfig = signingConfigs.getByName("debug")`, затем снова `.\gradlew.bat :app:assembleRelease`. Запасной вариант — подписать вручную через `jarsigner` и стандартный `%USERPROFILE%\.android\debug.keystore` (alias `androiddebugkey`, пароли `android`).

## Что проверить (чек-лист)

- [ ] Установка и первый запуск без краша
- [ ] Старт тренировки с главного экрана
- [ ] Несколько карточек подряд («Верно» / «Неверно» → «Следующая»)
- [ ] Смена **режима карточек** (random / mixed / new only и т.д.)
- [ ] Смена **языка интерфейса** и **текста на карточке**
- [ ] Работа **без интернета** (авиарежим)
- [ ] Работа **с интернетом** и загрузка картинок (если включены онлайн-источники и ключи при сборке)
- [ ] Поведение при **отсутствии картинки** (заглушка)
- [ ] **Настройки** (источники, онлайн, категории, разнообразие картинок)
- [ ] **Статистика**
- [ ] **Словарь**
- [ ] Внизу **Настроек** блок **«О сборке»**: версия, код, `debug` или `release`

## Как сообщить о баге

1. Модель телефона  
2. Версия Android  
3. Что делали перед ошибкой (шаги)  
4. Текст ошибки на экране или ожидание vs факт  
5. Скриншот или короткое видео  
6. Был ли интернет / Wi‑Fi  
7. Какие режимы в настройках (тренировка, картинки)  
8. Строку **«О сборке»** из приложения (версия · код · тип сборки)

## Секреты

- `local.properties` в git не коммитится (`.gitignore`). Ключи Pixabay/Pexels попадают в **APK как строки** при сборке с заполненным `local.properties` — для закрытого круга нормально; для публичного распространения ограничивайте ключи на стороне провайдера.

## Подпись

Сейчас **release** для теста может быть подписан **debug-keystore** (см. `app/build.gradle.kts`). Для Google Play нужен отдельный release-keystore; файлы `*.jks` / `*.keystore` не коммитить.
